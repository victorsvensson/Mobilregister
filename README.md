## About The Project

This project is a mobile registry which is a database of where you can see which employee that has which phone and phone number.

It has full Create Read Update Delete(CRUD) functionality.

![image](https://user-images.githubusercontent.com/80740775/226069498-a05e4d1f-18b4-4f1e-9944-995175a5ddf6.png)



<!-- USAGE EXAMPLES -->
## Usage

- Add, remove and edit phone models
- Add, remove and edit users
- Search for a user (by name)
- See list of a users Name, Phone number, Phone model. An edit button to edit a users information, and also if you want to remove a user. When you remove a user it will prompt before delete.

### Prerequisites

Make sure you have installed MongoDB and also mongosh
https://www.mongodb.com/try/download/community
https://www.mongodb.com/docs/mongodb-shell/

* npm
  ```sh
  npm install npm@latest -g
  ```


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/victorsvensson/Mobilregister.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
